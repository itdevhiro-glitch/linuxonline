# Floorbound: Ascension

Game web MMORPG turn-based dungeon floor dalam HTML, CSS, dan JavaScript murni.

## Cara Jalanin
1. Extract ZIP.
2. Buka `index.html` di browser.
3. Buat karakter: nama, job, dan ras.
4. Main dungeon, farming item, equip gear, jual-beli di trade market, dan push leaderboard.

## Fitur
- Job: Warrior, Archer, Mage, Support.
- Subclass unlock bertahap berdasarkan floor.
- Ras: Human, Dwarf, Demon, Ajin, Elf.
- Dungeon floor 1 sampai 1000.
- Boss tiap 10 floor.
- Rank F sampai SSS, dengan SSS mulai floor 500.
- Drop item class F sampai S.
- Weapon, armor, accessory bisa di-equip.
- Sistem trade: buy/sell item.
- Save otomatis pakai LocalStorage.
- Leaderboard lokal.

## Catatan
Ini versi frontend/local. Kalau mau multiplayer asli, market real-time, akun login, dan leaderboard global, tinggal dikembangkan pakai Firebase Auth + Realtime Database/Firestore.
